Praktis 6: Linear Inequalities in Two Variables

Page 1, Q1 (a-h): Draw and shade the regions that satisfy the inequalities.

Guidance for Drawing & Shading (Q1)

Step 1: Determine the Line Type. A solid line for `≥` and `≤`. A dashed line for `>` and `<`.

Step 2: Determine the Shading Region. For `y >` or `y ≥`, shade above. For `y <` or `y ≤`, shade below. For `x >` or `x ≥`, shade right. For `x <` or `x ≤`, shade left.

Page 1, Q2 (a-b): Determine which points satisfy (a) y > -x + 4 and (b) y < -x + 4.

Guidance for Identifying Points (Q2)

A point is in the region if its (x,y) coordinates make the inequality true.

(a) For y > -x + 4: The region above the line.

  • A(2,3): Is `3 > -2 + 4` (i.e., `3 > 2`)? Yes.
  • C(4,4): Is `4 > -4 + 4` (i.e., `4 > 0`)? Yes.

Points are A and C.

(b) For y < -x + 4: The region below the line.

  • B(-1,2): Is `2 < -(-1) + 4` (i.e., `2 < 5`)? Yes.
  • D(-2,5): Is `5 < -(-2) + 4` (i.e., `5 < 6`)? Yes.

Points are B and D.

Guidance for Writing Inequalities (Page 2, Q1)

(a) Line 1: Solid, x+y=4, above → x+y ≥ 4. Line 2: Solid, y=4, below → y ≤ 4. Line 3: Solid y-axis (x=0), right → x ≥ 0.

(c) Line 1: Solid, y=2x+4, below → y ≤ 2x+4. Line 2: Solid, y=x+2, above → y ≥ x+2. Line 3: Dashed, y=-2x+4, below → y < -2x+4.

(d) Line 1: Dashed, x=-2, right → x > -2. Line 2: Solid, y=x+2, below → y ≤ x+2. Line 3: Dashed, x+y=4, above → x+y > 4.

(f) Line 1: Solid, y=3x+3, below → y ≤ 3x+3. Line 2: Dashed, x+2y=6, above → x+2y > 6. Line 3: Dashed, 3y=x-6, above → 3y > x-6.
