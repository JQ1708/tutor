Page 1, Q1 (Measures of Dispersion): For the data 5, 8, 12, 14, 17, 19, 20, find all measures of dispersion.

Detailed Guidance

The data is sorted: 5, 8, 12, 14, 17, 19, 20. There are n = 7 data points.

  1. Range: Highest value - Lowest value = 20 - 5 = 15.
  2. Mode: The most frequent value. Since each value appears only once, there is No mode.
  3. First Quartile (Q1): Position is ¼(n+1) = ¼(8) = 2nd value. The 2nd value is 8.
  4. Median (Q2): Position is ½(n+1) = ½(8) = 4th value. The 4th value is 14.
  5. Third Quartile (Q3): Position is ¾(n+1) = ¾(8) = 6th value. The 6th value is 19.
  6. Interquartile Range (IQR): Q3 - Q1 = 19 - 8 = 11.
  7. Mean (μ):
    μ = (Sum of all values) / n
    Sum = 5+8+12+14+17+19+20 = 95. Mean = 95 / 7 ≈ 13.57.
  8. Variance (σ²):
    σ² = (Σx²) / n - μ²
    First, find the sum of squares (Σx²): 5²+8²+12²+14²+17²+19²+20² = 25+64+144+196+289+361+400 = 1479. σ² = (1479 / 7) - (13.57)² ≈ 211.29 - 184.14 = 27.15.
  9. Standard Deviation (σ):
    σ = √Variance
    σ = √27.15 ≈ 5.21.

Page 1, Q2 & Q3: Draw a dot plot and stem-and-leaf plots.

Guidance

For Dot Plot (Q2): Draw a number line from 1 to 5. Above the number 1, draw 3 dots. Above the number 2, draw 5 dots. Above the number 3, draw 9 dots, and so on. The number with the tallest stack of dots is the mode. The conclusion is that the data is concentrated around 3.

For Stem-and-Leaf Plot (Q3): The 'stem' represents the tens (or hundreds and tens) place, and the 'leaf' represents the ones place. For (a): Stem '8' would have leaves '2, 7, 9'. Stem '10' would have leaves '0, 2, 3, 5, 6, 7, 9, 9'. For (b): Stem '6' has leaves '2, 3, 7, 7, 9'. Stem '11' has leaves '0, 2'. Remember to include a key (e.g., 8 | 2 means 82).

Page 3, Q4: Find measures for the frequency data.

Detailed Guidance (Frequency Data)

Total Frequency (Σf) = 5+12+14+15+12+8 = 66.

  1. Median: Position is (66+1)/2 = 33.5th value. Use cumulative frequency: 5, 17, 31, 46... The 33.5th value lies in the group where the number is 6.
  2. Mean: Calculate Σfx = (3×5) + (4×12) + (5×14) + (6×15) + (7×12) + (8×8) = 15+48+70+90+84+64 = 371. Mean = Σfx / Σf = 371 / 66 ≈ 5.62.
  3. Variance: Calculate Σfx² = (3²×5) + (4²×12) + ... + (8²×8) = 45+192+350+540+588+512 = 2227. σ² = (Σfx² / Σf) - μ² = (2227 / 66) - (5.62)² ≈ 33.74 - 31.58 = 2.16.

Page 4, Q12: Determine if there are outliers in 12, 12, 13, 15, 16, 17, 18, 20, 22, 23, 33.

Detailed Guidance (Outliers)

  1. Find Quartiles: Data is sorted, n=11. Q1 is the 1/4(12) = 3rd value = 13. Q3 is the 3/4(12) = 9th value = 22.
  2. Calculate IQR: IQR = Q3 - Q1 = 22 - 13 = 9.
  3. Find Fences: Lower Fence = Q1 - 1.5 * IQR = 13 - 1.5 * 9 = 13 - 13.5 = -0.5. Upper Fence = Q3 + 1.5 * IQR = 22 + 1.5 * 9 = 22 + 13.5 = 35.5.
  4. Check data: Is any value less than -0.5 or greater than 35.5? No. Therefore, there are no outliers.

Page 7, Q6: Data 21, 24, 26, 28, x, 29, 26, 25, 24 has a mean of 26. Find x.

Detailed Guidance (Finding Unknown)

  1. Write the mean formula: Mean = (Sum of values) / n.
  2. Identify knowns: Mean = 26, n = 9.
  3. Sum the known values: 21+24+26+28+29+26+25+24 = 203.
  4. Set up the equation: 26 = (203 + x) / 9.
  5. Solve for x: 26 × 9 = 203 + x. 234 = 203 + x. x = 234 - 203 = 31.

Page 8, Q9 (Median): The sorted data 9, 12, 13, x, y, 19, 23, 25, 26 has a median of 17. Find possible (x, y) where x < y.

Detailed Guidance (Median with Unknowns)

  1. Find the median position: There are n=9 data points. The median is the (9+1)/2 = 5th value.
  2. Identify the median value: Counting along the list, the 5th value is 'y'.
  3. Equate and find y: Since the median is 17, we have y = 17.
  4. Determine the range for x: The data is sorted, so 'x' must be between the 4th value (13) and the 5th value (y=17). Also, x < y. So, 13 ≤ x < 17.
  5. List possible integer values for x: x can be 13, 14, 15, or 16.

Page 9, Q14 (Mean with variables): 3, 4, y, 6, 11, x has a mean of 6. Given x = 3y, find x and y.

Detailed Guidance (Simultaneous Equations)

  1. Use the mean to form Equation 1: Mean = (3+4+y+6+11+x) / 6 = 6. 24 + x + y = 36. x + y = 12.
  2. Use the given relationship as Equation 2: x = 3y.
  3. Substitute Eq. 2 into Eq. 1: (3y) + y = 12. 4y = 12. y = 3.
  4. Substitute y back into Eq. 2 to find x: x = 3 * (3). x = 9.
