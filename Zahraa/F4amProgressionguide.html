12.1 Arithmetic Progression

3. Find the number of terms of the arithmetic progression 4, 1, -2, ..., -23.

Step 1: Identify the first term (a), common difference (d), and the last term (Tn).
a = 4
d = 1 - 4 = -3
Tn = -23
Step 2: Use the formula Tn = a + (n-1)d to find n.
-23 = 4 + (n-1)(-3)
-27 = (n-1)(-3)
9 = n - 1
n = 10

4. Find the number of the multiples of 3 between 20 and 110.

Step 1: Identify the first term (a), common difference (d), and the last term (Tn) in the sequence.
First multiple of 3 after 20 is a = 21.
Last multiple of 3 before 110 is Tn = 108.
The common difference is d = 3.
Step 2: Use the formula Tn = a + (n-1)d to find n.
108 = 21 + (n-1)(3)
87 = (n-1)(3)
29 = n - 1
n = 30

5. For the arithmetic progression 5, 8, 11, ..., which term is equal to 320?

Step 1: Identify a, d, and Tn.
a = 5
d = 8 - 5 = 3
Tn = 320
Step 2: Use the formula Tn = a + (n-1)d to find n.
320 = 5 + (n-1)(3)
315 = (n-1)(3)
105 = n - 1
n = 106. It is the 106th term.

6. Given that 4k, 12, 2k2+4k are three consecutive terms of an arithmetic progression, find the possible values of k.

Step 1: Use the property of an AP: T2 - T1 = T3 - T2.
12 - 4k = (2k2 + 4k) - 12
Step 2: Rearrange the terms to form a quadratic equation.
0 = 2k2 + 8k - 24
k2 + 4k - 12 = 0
Step 3: Factorise the quadratic equation to find the values of k.
(k + 6)(k - 2) = 0
k = -6, k = 2

7. The 12th term and the 17th term of an arithmetic progression are 38 and 53 respectively. Find the common difference.

Step 1: Set up two equations using Tn = a + (n-1)d.
T12 = a + 11d = 38 --- (1)
T17 = a + 16d = 53 --- (2)
Step 2: Solve the simultaneous equations.
(2) - (1): (a + 16d) - (a + 11d) = 53 - 38
5d = 15
d = 3

8. For the arithmetic progression 184, 181, 178, ..., find the value of the first negative term.

Step 1: Identify a and d. Then set up an inequality Tn < 0.
a = 184, d = 181 - 184 = -3
a + (n-1)d < 0
184 + (n-1)(-3) < 0
Step 2: Solve the inequality for n.
184 - 3n + 3 < 0
187 < 3n
n > 1873 or 62.33
Step 3: Find the smallest integer n that satisfies the inequality and find Tn.
The smallest integer n is 63.
T63 = 184 + (63-1)(-3) = 184 - 186 = -2

9. Find the value of the first positive term of the arithmetic progression -101, -97, -93, ...

Step 1: Identify a and d. Then set up an inequality Tn > 0.
a = -101, d = -97 - (-101) = 4
-101 + (n-1)(4) > 0
Step 2: Solve for n.
4(n-1) > 101
n-1 > 25.25
n > 26.25
Step 3: Find the smallest integer n and then find the term Tn.
The smallest integer n is 27.
T27 = -101 + (27-1)(4) = -101 + 104 = 3

10. If x+4, 2x+5 and 2x2+4 are three consecutive terms of an arithmetic progression, find the possible values of x.

Step 1: Use the property T2 - T1 = T3 - T2.
(2x+5) - (x+4) = (2x2+4) - (2x+5)
Step 2: Simplify and form a quadratic equation.
x + 1 = 2x2 - 2x - 1
2x2 - 3x - 2 = 0
Step 3: Factorise and solve for x.
(2x + 1)(x - 2) = 0
x = -12, x = 2

11. Find the number of terms in the arithmetic progression 2p+3, 6p+2, 10p+1, ..., 30p-4.

Step 1: Find the first term (a) and common difference (d).
a = 2p+3
d = (6p+2) - (2p+3) = 4p-1
Tn = 30p-4
Step 2: Use the formula Tn = a + (n-1)d.
30p-4 = (2p+3) + (n-1)(4p-1)
Step 3: Solve for n.
28p - 7 = (n-1)(4p-1)
7(4p - 1) = (n-1)(4p-1)
7 = n - 1
n = 8

12. Find the sum of the first 12 terms of the arithmetic progression -10, -7, -4, ...

Step 1: Identify a, d, and n.
a = -10
d = -7 - (-10) = 3
n = 12
Step 2: Use the sum formula Sn = n2[2a + (n-1)d].
S12 = 122[2(-10) + (12-1)(3)]
S12 = 6[-20 + (11)(3)]
S12 = 6[-20 + 33] = 6(13) = 78

13. Find the sum of the first n terms of the arithmetic progression 11, 7, 3, ...

Step 1: Identify a and d.

14. For the arithmetic progression -4, 1, 6, ..., find the sum of all the terms from the 6th term to the 14th term.

Step 1: Understand the question. We need to find S14 - S5.
The sum from the 6th to 14th term is the sum of the first 14 terms minus the sum of the first 5 terms.
Step 2: Identify a and d. Calculate S14.
a = -4, d = 5
S14 = 142[2(-4) + (14-1)(5)] = 7[-8 + 65] = 7(57) = 399
Step 3: Calculate S5 and find the final answer.
S5 = 52[2(-4) + (5-1)(5)] = 2.5[-8 + 20] = 2.5(12) = 30
Sum = S14 - S5 = 399 - 30 = 369

16. How many terms of the arithmetic progression 12, 16, 20, ... must be taken for the sum to be equal to 132?

Step 1: Identify a, d, and Sn. Set up the sum equation.

n2 + 5n - 66 = 0
Step 3: Factorise and solve for n.

17. The first term and the last term of an arithmetic progression are -12 and 36 respectively. If the sum of all the terms is 240, find the number of terms.

Step 1: Identify a, l (last term), and Sn.
a = -12, l = 36, Sn = 240
Step 2: Use the sum formula Sn = n2(a + l).
240 = n2(-12 + 36)
240 = n2(24)
240 = 12n
n = 20

18. The second term of an arithmetic progression is 8 while the sum of the first six terms is 30. Find the common difference.

Step 1: Set up two equations from the given information.
T2 = a + d = 8 --- (1)
S6 = 62[2a + 5d] = 30 --- (2)
Step 2: Simplify equation (2) and solve simultaneously with (1).
From (2): 3[2a + 5d] = 30 => 2a + 5d = 10
From (1): a = 8 - d. Substitute into the simplified (2):
2(8 - d) + 5d = 10
16 - 2d + 5d = 10
3d = -6 => d = -2

19. The third term and the seventh term of an arithmetic progression are 6 and 26 respectively. Find the eleventh term.

(2)-(1): 4d = 20 => d = 5
Sub into (1): a + 2(5) = 6 => a = -4
Step 2: Calculate the 11th term (T11).
T11 = a + 10d = -4 + 10(5) = -4 + 50 = 46

20. The sum of the first 6 terms of an arithmetic progression is 72 while the sum of the first 9 terms is 81. Find the common difference.

26. In an arithmetic progression, the sum of all the terms from the second term to the fourth term is 18. If the first term is 3, find the common difference.

Step 1: Express the sum in terms of a and d.
T2 + T3 + T4 = 18
(a+d) + (a+2d) + (a+3d) = 18
3a + 6d = 18
Step 2: Substitute a=3 and solve for d.
3(3) + 6d = 18
9 + 6d = 18
6d = 9 => d = 1.5

27. An arithmetic progression has 16 terms. The first term is 5. Given that the sum of the last 6 terms is 255, find the common difference.

Step 1: Express the sum of the last 6 terms.
The last 6 terms are T11 to T16. Their sum is S16 - S10 = 255.
Step 2: Write out the sum equations and substitute a=5.
S16 = 162[2(5) + 15d] = 8(10 + 15d) = 80 + 120d
S10 = 102[2(5) + 9d] = 5(10 + 9d) = 50 + 45d
Step 3: Solve for d.
(80 + 120d) - (50 + 45d) = 255
30 + 75d = 255
75d = 225 => d = 3

28. If the sum of the first n terms of an arithmetic progression is given by Sn=n(4n-7), find the fourth term.

Step 1: Understand the relationship between Tn and Sn.
Tn = Sn - Sn-1. We need to find T4.
Step 2: Calculate S4 and S3.
S4 = 4(4x4 - 7) = 4(16 - 7) = 4(9) = 36
S3 = 3(4x3 - 7) = 3(12 - 7) = 3(5) = 15
Step 3: Calculate T4.
T4 = S4 - S3 = 36 - 15 = 21

30. The sum of the first n terms of an arithmetic progression is given by Sn=12n(3n+2). Calculate the eighth term.

42. If x+2, 2x+4 and 5x+4 are three consecutive terms of a geometric progression, find the possible values of x.
