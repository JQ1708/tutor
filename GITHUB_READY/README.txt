Upload this package to GitHub (simple method):

1) In your GitHub repository page, click "Add file" -> "Upload files".
2) Open this local folder:
   GITHUB_READY
3) Drag the entire folder named "docs" into GitHub upload area.
4) Click "Commit changes".

Enable GitHub Pages:
1) Repository -> Settings -> Pages
2) Source: "Deploy from a branch"
3) Branch: main
4) Folder: /docs
5) Save

Your colleagues can open:
https://<your-username>.github.io/<your-repo>/
