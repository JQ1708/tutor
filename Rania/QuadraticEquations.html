Solving by Factorising

How do I solve by factorising?

This is often the quickest method. The goal is to put the equation into two brackets.

  1. Rearrange the equation into the form ax² + bx + c = 0.
  2. Factorise the quadratic expression.
  3. Set each bracket equal to zero and solve for x. Why? Because if two things multiply to make zero, one of them must be zero!

Example 1: Simple Case

Solve: (x - 3)(x + 7) = 0

First bracket: x - 3 = 0 → x = 3

Second bracket: x + 7 = 0 → x = -7

The solutions are the numbers in the brackets but with the opposite signs.

Example 2: Harder Case

Solve: (2x - 9)(8x - 5) = 0

First bracket: 2x - 9 = 0 → 2x = 9 → x = 9⁄2

Second bracket: 8x - 5 = 0 → 8x = 5 → x = 5⁄8

The Quadratic Formula

When do I use the formula?

This is your go-to method when a quadratic won't factorise easily. It works for any quadratic equation!

First, make sure the equation is in the form ax² + bx + c = 0.

The Discriminant: b² - 4ac

The part under the square root, called the discriminant, tells you how many solutions you'll get.

  • If b² - 4ac > 0 (positive): You get TWO real solutions.
  • If b² - 4ac = 0: You get just ONE real solution (a repeated root).
  • If b² - 4ac < 0 (negative): You get NO real solutions (you can't square root a negative!).

Completing the Square

What is completing the square?

It's a way of rewriting a quadratic ax² + bx + c into the form a(x + p)² + q. This form is useful for finding the minimum or maximum point of the graph, and for solving the equation.

How to do it (when a = 1)

Let's rewrite x² + 6x + 5:

  1. Halve the 'b' term: 6 ÷ 2 = 3. This is 'p'. So we have (x + 3)².
  2. Expand this bracket: (x + 3)² = x² + 6x + 9.
  3. Compare and adjust: We want x² + 6x + 5, but we have x² + 6x + 9. We need to subtract 4.
  4. Final Form: (x + 3)² - 4.

How to solve using this form

Solve (x + 3)² - 4 = 0:

  1. Add 4 to both sides: (x + 3)² = 4
  2. Square root both sides (remember ± !): x + 3 = ±2
  3. Solve for x:
    • x + 3 = 2 → x = -1
    • x + 3 = -2 → x = -5

Which Method Should I Use?

Your Decision Checklist

Follow this order to pick the best method for the job.

Step 1: Is the equation actually a quadratic? It must have an x² term.
Step 2: Does the question specifically tell you to use a certain method (e.g., "solve by completing the square")? If yes, you must use that method.
Step 3: Can you factorise it quickly? Always try this first! It's usually the fastest way.
Step 4: If you can't see how to factorise it, or it looks complicated, use the Quadratic Formula. It's reliable and always works.

Examiner's Tips!

  • Always make sure your equation is in the form ax² + bx + c = 0 before you start.
  • Be careful with negative signs, especially when using the formula. Put negative numbers in brackets in your calculator, e.g., (-5)².
  • Always give your answers to the required degree of accuracy (e.g., 2 decimal places).
